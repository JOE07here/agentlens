AgentLens — Architecture v0.2

Rebuilt as a browser-only web app. Same analysis pipeline, now running entirely client-side — and with cross-stack correlation (your moat) promoted to its own stage.

Version: 0.2 — supersedes v0.1 Owner: Joe Stack: Next.js + TypeScript Build with: Claude Code
What changed from v0.1: form factor is now a web app (not a CLI), the engine is TypeScript (not Python), and all processing runs in the browser (not on a server). Everything else — the pipeline, the canonical model, the growth path — carries over. The competitor research also promoted cross-stack correlation to a first-class stage, since it's the one thing nobody else does.

This is the cross-stack view no competitor offers — make it the product's headline.

03Components (TypeScript modules)

① Source adapters

midpointAdapter parses the XML/CSV export; keycloakAdapter parses the JSON realm export. Both run in-browser via a parser like fast-xml-parser.

② Normalizer + classifier

Folds both into the canonical model and tags human vs non-human/agent with configurable patterns.

③ Correlator — the moat

Links a MidPoint identity to its Keycloak client into one logical agent. The differentiator; give it real care.

④ Rule engine

Pluggable rules over the model: over-privilege, dormancy, SoD conflict, standing credential — each emits findings with a plain reason.

⑤ Scorer

Levels each finding (High/Med/Low) and ranks them worst-first.

⑥ Report view + export

A React view with risk badges; export to standalone HTML (browser print → PDF) and JSON/CSV.
